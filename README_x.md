# 📗 Cẩm nang Huấn luyện Smol — Bản dịch Tiếng Việt

> Bản dịch tiếng Việt của **"The Smol Training Playbook"** bởi HuggingFace — tài liệu chi tiết về hành trình xây dựng SmolLM3, mô hình ngôn ngữ 3B tham số đạt hiệu suất đẳng cấp thế giới.

## 🌐 Xem trực tuyến

👉 **[tuandung222.github.io/smol-vi](https://tuandung222.github.io/smol-vi/)**

## 🛠️ Phát triển cục bộ

```bash
# Cài đặt dependencies
npm install

# Chạy development server
npm start

# Build production
npm run build
```
